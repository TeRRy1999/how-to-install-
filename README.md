

1. 安裝 ibus  鍵盤管理工具

 sudo apt-get install ibus ibus-gtk ibus-gtk3 ibus-qt4

2. 安裝 新酷音輸入法

 sudo apt-get install ibus-chewing

3.轉換成  ibus 鍵盤管理

   im-switch -s ibus

4.設定 ibus 支援的輸入法

   ibus-setup

 

input method -> customize active input methods (勾選)

select an input method -> 選取新酷音後按下右邊的 Add

之後就可以透過按下　ctrl + space 切換輸入法
